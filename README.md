# CLI-Based-Personal-Finance-Manager

A lightweight and intuitive Command-Line Interface (CLI) tool to track, manage, and analyze your personal finances, including income, expenses, and monthly summaries.
Personal Finance Management CLI is a simple yet powerful Python-based command-line application designed to help users manage their money efficiently.

Features:
Add, view, and delete transactions
Track expenses and income by category
Generate monthly summaries and balance reports
Persist financial data in a JSON file
Operate entirely from the terminal for quick and distraction-free finance management

Instalation:
clone the repository: git clone https://github.com/Saria-Rukhsar/finance-manager.git
navigate to the project diroctory: cd finance-manager

How to Run:
Run the CLI application using: python main.py
